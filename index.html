Bienvenidos al tutorial de CoCo2d-x.

Este tutorial sirve para la instalación de coco2d-x para windows y uso en Visual Studio

Este tutorial tiene como proposito explicar el uso de sprites en coco2d-x asi como algunos ejemplos basicos

Prequisitos

  • Sistema Operativo Windows 7 o posterior
  • Visual Studio 2013 o posterior
  • Compilador Python 2.X

Python se puede descargar desde el siguiente enlace https://www.python.org/downloads/

Instalación

  1. Descargamos Coco2d-x desde la pagina original o si se desea ver Github. Despues de esto descomprimimos y lo guardamos en donde queramos.

  2. Usado los símbolos de sistemas (cmd) procedemos a buscar la carpeta donde guardamos, y ejecutamos "Python setup.py". Tambien pueden ir manualmente usando el explorador y ejecutarla setup.

  3. Una vez que se ejecuto, cerramos la carpeta o el cmd y volvemos a abrir otro cmd, esto es ya que pide la restauración del cmd para utilizar el nuevo comando "cocos". Una vez que ejecutamos cocos, nos pedira una verificación si es la primera vez que lo ejecutamos, escogemos [y]es y ya esta listo para utilizarse.

Creación del primer programa, ejemplo de hello world

  1. Accedemos nuevamente a cmd y ejecutamos "cocos new -l cpp nombreproyecto -d C:..." donde '-l cpp' se declara que es un archivo de c++ con proyecto llamado 'nombreproyecto', y el '-d C:...' escogemos en donde queremos que este se guarde. Si se quiere trabaja con proyectos tipo javascript o lua declaramos 'js' o 'lua' en vez de 'cpp'.

  2. Ya que termino de crear el proyecto, abrimos la ubicacion de la carpeta destino con el explorador y nos direjimos a la carpeta "coco2d" y posteriormente a la carpeta "proj.win32", ahi encontramos archivos ejecutables para VisualStudio (VS), y ejecutamos "nombreproyecto.sln"

3.- Una vez dentro de VS simplemente usamos el boton "local windows debugger" o simplemente "debugger" (el nombre del botón depende de la version del VS, de la configuracion de), tambien puedes precionar F5, una vez hecho esto confirmamos que queremos ejecutar las clases mencionadas y esperamos a que compile, puede tardar hasta media hora e incluso pediera descargar algunos componentes.

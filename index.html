Bienvenidos al tutorial de CoCo2d-x.

Este tutorial sirve para la instalación de coco2d-x para Windows usando Visual Studio y lenguaje C++ y tiene como proposito explicar la instalación, el uso de sprites en coco2d-x asi como otros ejemplos basicos.

Prequisitos

  • Sistema Operativo Windows 7 o posterior
  • Visual Studio 2013 o posterior
  • Compilador Python 2.X

Python se puede descargar desde el siguiente enlace https://www.python.org/downloads/

Instalación

  1. Descargamos Coco2d-x desde la pagina original o si se desea ver Github. Despues de esto descomprimimos y lo guardamos en donde queramos.

  2. Usado los símbolos de sistemas (cmd) procedemos a buscar la carpeta donde guardamos, y ejecutamos el comando "Python setup.py". Tambien pueden ir manualmente usando el explorador y ejecutarlo.

  3. Una vez que se ejecuto, cerramos la carpeta o el cmd y volvemos a abrir otro cmd, esto es ya que pide la restauración del cmd para utilizar el nuevo comando "cocos". Una vez que ejecutamos cocos, nos pedira una verificación si es la primera vez que lo ejecutamos, escogemos [y]es y ya esta listo para utilizarse.

Creación del primer programa, ejemplo de hello world

  1. Accedemos nuevamente a cmd y ejecutamos "cocos new -l cpp nombreproyecto -d C:\..." donde '-l cpp' se declara que es lenguaje c++ con proyecto un llamado 'nombreproyecto', y el '-d C:\...' escogemos en donde queremos que este se guarde. Si se quiere trabajar con proyectos tipo javascript o lua declaramos 'js' o 'lua' en vez de 'cpp'.

  2. Ya que termino de crear el proyecto, abrimos la ubicacion de la carpeta destino con el explorador y nos direjimos a la carpeta "proj.win32", ahi encontramos archivos ejecutables para VisualStudio (VS), y ejecutamos "nombreproyecto.sln"

  3. Una vez dentro de VS simplemente usamos el boton "local windows debugger" o simplemente "debugger" (el nombre del botón depende de la version del VS, de la configuracion, etc), tambien puedes precionar F5, una vez hecho esto confirmamos que queremos ejecutar las clases mencionadas y esperamos a que compile. Puede tardar hasta media hora e incluso pediera descargar algunos componentes pero esto solo ocurre la primera vez, y las siguentes veces no tardara en compilar.

Ahora tenemos nuestro ejemplo de hola mundo de Coco2d-x
